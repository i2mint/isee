isee.common¶

isee.common.get_env_var(key)[source]¶

Get the value of an environment variable. If the variable is not defined or is empty, raise a RuntimeError.

Parameters

key – The name of the environment variable.

Returns

The value of the environment variable.

Raises

RuntimeError – If the environment variable is not defined or is empty.

>>> import os
>>> os.environ['TEST'] = 'test'
>>> get_env_var('TEST')
'test'

>>> get_env_var('TEST2')
Traceback (most recent call last):
...
RuntimeError: TEST2 is not defined or is empty!

>>> os.environ['TEST3'] = ''
>>> get_env_var('TEST3')
Traceback (most recent call last):
...
RuntimeError: TEST3 is not defined or is empty!
