isee.file_modification_utils

This module provides utilities for updating files.

Functions: - update_helm_tpl: Update the Helm chart template files. - update_manifest: Update the manifest file. - update_setup_cfg: Update the setup.cfg file. - update_setup_py: Update the setup.py file. - _get_setup_filepath: Get the setup file path. - _update_file: Update the file content.

isee.file_modification_utils.update_setup_cfg(*, file_path: str = 'setup.cfg', version: str | None = None)[source]

Update the version in setup.cfg file.

Parameters:

  • file_path – Path to the setup.cfg file

  • version – The version to set. If None, tries to get it from the VERSION environment variable.
